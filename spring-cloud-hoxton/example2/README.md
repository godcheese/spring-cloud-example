# example2

Spring Cloud Eureka 服务注册中心实例，简单的服务注册实现。

- 支持负载平衡。
- 支持 Eureka Server 用户名密码安全认证。

```
│
├── example1
│   │  
│   ├── eureka-server ------------ Eureka Server 服务注册中心
│   ├── service1 ----------------- 服务 1
│   ├── service2 ----------------- 服务 2
│   ├── service2-1 --------------- 服务 2 的 copy 1
│   ├── consumer1 ---------------- 消费者 1
│   │ 
```
